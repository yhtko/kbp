# AutoSignage for Kintone


一覧画面をサイネージ化するプラグインです。


## 機能
- サイネージモード：ヘッダー/メニュー非表示、全画面風、スクロールバー非表示
- 自動スクロール：一定速度で下方向へ、最下部で停止→先頭へ、指定回数で自動リロード
- 自動リロード：間隔指定＋±5秒ジッター
- URLパラメータ上書き：`?kiosk=1&interval=180&speed=50&stop=2&loops=5&zoom=1.2&theme=dark`
- 表示オプション：ズーム倍率、テーマ（ライト/ダーク）


## インストール
1. `npm i`
2. `npm run build` → `dist/` 配下に出力（本構成では生ファイルでもOK）
3. `npm run pack` で `plugin.zip` を作成
4. kintone 管理からプラグインをアップロード→アプリに適用


## 設定
- 設定画面で各項目を入力し保存
- 一時的に切り替えたい場合はURLパラメータで上書き可能


## 注意
- ブラウザの完全なフルスクリーンはユーザー操作が必要です（F11 または追加ボタン）。
- ズームは `documentElement.style.zoom` を使用。環境によっては表示の差異があります。
- セキュリティ方針：`eval/new Function/innerHTML(動的)` 不使用。DOM APIで値設定。
